# Vendor payments data

Place **`Vendor-Payments_2021-23_FY_2022_.csv`** in this folder.

Download the 2021–23 biennium vendor payments from [Washington State Open Checkbook](https://fiscal.wa.gov/Spending/Checkbook2123) (Download Data), or export the **FY 2022** sheet to CSV.

Then regenerate app aggregates:

```bash
npm run generate-data
```
